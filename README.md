# BYT( Bring Your own Token)
## Tech Stack
### Web
- Next
- Drizzle
- Postgres
- Shadcn
### Mobile
- Expo
  Will consume data from the Next Server on the web
### Desktop
- Tauri(Rust)
- Frontend Next
  Will also comsume data from the Next Server on web

## Guide to get Started



```git
git clone https://github.com/kelvinwambua/byt
```
```
 cd byt
```

## Guide to run specific apps

### Web

```cd
cd web
```

```pnpm
pnpm install
```
```pnpm
pnpm run dev
```

### Mobile
You will probably need an emulator or wireless/usb debugging on your device along with android sdk tools and java 17 installed 
```cd
cd mobile
```

```pnpm
  pnpm install
```
```pnpm
npx expo run
```
### Desktop
Make sure you have Rust Installed
```cd
cd desktop
```

```pnpm
  pnpm install
```
```pnpm
pnpm tauri dev
```
